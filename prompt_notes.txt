## The App
**SplitFree** — A trip expense splitting web app (like Splitwise but free, group-trip focused, and invite-link based). Single HTML file, Firebase backend, hosted on GitHub Pages.

## Decisions Made
- No unequal splits — group trips split equally, include/exclude only
- No multi-currency conversion — out of scope
- Admin = trip creator, not based on sign-in order
- Anyone can mark settlement done (not just admin)
- Only marker (or admin) can undo a settlement
- Archived trips hidden by default, toggle to show
- Delete trip fully cleans Firestore — no orphan data
